## 使用说明

使用 `find-skills` skill 查找和安装 skill，具体命令见各 skill 的介绍页面。

### 飞书 Skill 更新

飞书官方 skill（lark-* 系列）通过 `lark-cli` 统一更新：

```bash
lark-cli update
```

此命令会**同时更新 CLI 和所有飞书 Skills**。更新后需重新打开 AI Agent 以加载最新版本。

### 全量校验

当需要校验 skills 目录与 README 目录一致性时，告诉 agent "校验 skill 目录" 即可。
