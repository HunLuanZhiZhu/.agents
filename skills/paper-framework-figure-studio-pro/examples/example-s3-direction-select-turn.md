# Example: S3-DIRECTION-SELECT

Run only S3. Select the strongest structure direction from the S2 sketches and define local refinement needs. Stop with an S4-CANDIDATE-BRIEF next-step prompt that has a shared part plus two branches: branch A keeps later reference images hand-drawn / sketch-like; branch B makes later reference images easier to redraw manually as SVG/PPT with clean geometry. Branch B is the default if the user does not choose.
